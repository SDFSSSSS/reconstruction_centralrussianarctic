# reconstruction_centralrussianarctic
Data and scripts for Reconstruction of Near Surface Summer Air Temperature in the Central Part of the Russian Arctic over the past 2000 Years.  
Region of interest: Western - 60E-100E, Eastern - 100E-140E, latitude boundaries - 65N-80N  
Temporal resolution: 1 year  
## Data:	
1. Script for all the steps for reconstruction with comments
2. reconstruction_code.zip - archive with all needed data for working script
3. Table with the sources for all used data
## Extra information:
Borehole series reference: Pollack H., Demezhko D., Duchkov A., Golovanova I., Huang S., Shchapov V., Smerdon J. Surface temperature trends in Russia over the past five centuries reconstructed from borehole temperatures // Journal of Geophysical Research: Solid Earth. – 2003. – Т. 108, № B4.  
NOAA map with proxies: https://www.ncei.noaa.gov/maps/paleo/  
Big NOAA archive: https://www.ncei.noaa.gov/pub/data/paleo/
